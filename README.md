# GameOfThree backend

This repository contains a simple Node.js application.

## Build Setup

``` bash
# install dependencies
npm ci

# start server
npm start
```

## Server

The server is hosted here: [https://game-of-3-backend.herokuapp.com/](https://game-of-3-backend.herokuapp.com/)
